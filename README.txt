# ПРОГРАММА ДЛЯ СОЗДАНИЯ ШКОЛЬНЫХ РАСПИСАНИЙ ПО ЗАДАННЫМ УСЛОВИЯМ НА PYTHON

## Требования
- Python 3.8+
- IDE для работы с Python

## Начало работы
- В папке data создать текстовый файл для каждого класса в формате '[параллель класса][буква класса].txt'.
  Пример: 9А.txt
- В файле каждого класса указать имя учителя, предмет и количество часов в неделю строго через запятую и пробел заглавными буквами.
  ВАЖНО: в названиях предметов пробел необходимо заменить на _
  Пример: ИВАНОВА И.А., РУССКИЙ_ЯЗЫК, 5
          СМИРНОВ А.В., АЛГЕБРА, 4
- Открыть файл subject.py в IDE и записать списком все возможные предметы заглавными буквами, сохранить файл.
  Пример: 
    class Subject(Enum):
        РУССКИЙ_ЯЗЫК = 1
        ЛИТЕРАТУРА = 2
        АЛГЕБРА = 3
        ГЕОМЕТРИЯ = 4
        ФИЗИКА = 5
        ХИМИЯ = 6
  ВАЖНО: необходимо сохранить нумерацию по порядку.
- Запустить файл main.py в IDE
  После выполнения программы полное расписание будет выведено в папку schedules в файл 'full schedule.txt'.
  Расписания классов будут выведены в папку schedules, каждое в свой файл.
